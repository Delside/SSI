## About The Project
SSI is frontend part of MEAN stack project Słownik Skrótów Internetowych. Build with Angular 10, connected to NodeJS and MongoDB backend.

### Build With
* Angular 10
* Bootstrap 5
* ngx-toastr
* VS Code IDE

### Functionality
SSI UI allows everyone to view all Interned slang words, that are in the MongoDB database. Using Angular 10 and Bootstrap
to provide modern looking website. Users can add their own words with special form on seperate subpage.

## Getting Started
For local usage I recommend cloning the repo with command:
```sh
git clone https://github.com/Delside/SSI.git
```
You will need Browser and IDE with Angular 10 to run the project.

## Backend

For proper functioning, you will also need backend part of project. It is available here: [SSI - API](https://github.com/Delside/SSI-API)
